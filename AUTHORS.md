# Authors and contributors

This project is a generalized, anonymized distillation of the
standards-based grading (SBG) system used in **MAT188 (Linear Algebra,
University of Toronto)**.

## Lead author

**Camelia Karimianpour** — University of Toronto
Course design, learning-standards framework, labeling scheme, SBG
methodology, and overall direction. Credited on the project as a whole.

## Contributors

**Simeon Wong** — Analysis pipeline (Python)
Original learning-standards analysis pipeline that the converters and
evaluator in `scripts/` are adapted from
([dtxe/mat188_learning_standards](https://github.com/dtxe/mat188_learning_standards)).
Credit Simeon for the data pipeline and any Python-derived work.

**Samuel P. Dumas** — LaTeX system
Co-author of the MAT188 standards-based LaTeX environment (style
packages, document templates, and the original documentation). Credit
Sam for the LaTeX system and any LaTeX-derived work. *(The LaTeX system
lives in the companion `MAT188-Restructured` repository; this starter
kit reuses its standards/labeling conventions.)*

## How credit maps to the material

| Part of the project | Credit |
|---|---|
| Everything / overall design | Camelia Karimianpour |
| Python pipeline, converters, evaluator, notebook | Simeon Wong (with Camelia) |
| LaTeX style packages, readings, worksheets, doc templates | Samuel P. Dumas (with Camelia) |
